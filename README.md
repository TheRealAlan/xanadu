# Xanadu
A text-based game engine.

View a demo [here](http://octopress.org/docs/deploying/github/).

### Get the Repo
``` bash
git clone https://github.com/TheRealAlan/xanadu.git
```

### Get the Gems
``` bash
bundle install
```

### Get the Bower Packages
``` bash
bower install
```

### Run It
``` bash
middleman server
```
